# LifeCompass — FIRE・退職資産シミュレーター

NISA・iDeCo・特定口座・現金を統合し、FIREや老後の資産計画をシミュレートするブラウザ完結型ツールです。

## 主な機能

- **口座別シミュレーション** — NISA・iDeCo（一時金/年金受取）・特定口座・現金を個別に追跡
- **モンテカルロ法** — 市場変動を確率的にシミュレートし、破綻確率を計算
- **ポートフォリオ自動計算** — 資産クラス配分からμ・σを自動算出（LTCMA 2026準拠）
- **ライフイベント対応** — 住宅購入・教育費・退職金などを任意のタイミングで設定
- **改善案インパクト比較** — 支出削減・退職延長・余剰CF投資などの効果を定量表示
- **AI分析** — Gemini APIによる資産計画のコメント生成
- **プロファイル管理** — 複数プロファイルの保存・比較・URLシェア

## 使い方

1. `simulator.html` をブラウザで開く（ローカルファイルとして動作・サーバー不要）
2. 基本情報・収支・口座残高を入力する
3. 「固定計算」または「モンテカルロ実行」ボタンで結果を表示

## AI分析機能を使う場合

AI分析には **Google Gemini APIキー** が必要です（無料枠あり）。

1. [Google AI Studio](https://aistudio.google.com/) でAPIキーを取得
2. ツール内の「AI分析」セクションでAPIキーを入力（ブラウザのlocalStorageに保存・外部送信なし）

> **APIキーを取得しない場合でも、シミュレーション機能はすべて利用できます。**

## 技術仕様

- **動作環境** — モダンブラウザ（Chrome / Firefox / Safari / Edge）
- **依存ライブラリ** — [Chart.js 4.4.1](https://www.chartjs.org/)・[Tabler Icons](https://tabler.io/icons)（CDN読み込み）
- **バックエンドなし** — 全計算をブラウザ内で完結。データは外部サーバーへ送信されません
- **プロファイル保存** — ブラウザのlocalStorageを使用
- **積立タイミング** — 年末積立方式で計算します（前年末残高を1年運用後、その年の積立を加算）。年初一括積立より保守的な推定になります
